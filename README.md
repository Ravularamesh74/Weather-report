# Weather Report

A comprehensive weather report application.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Navigate to the project directory:
   ```sh
   cd Weather-report
   ```

2. Install the necessary dependencies:
   ```sh
   npm i
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

You can deploy this project using any modern web hosting service like Vercel, Netlify, or GitHub Pages.
